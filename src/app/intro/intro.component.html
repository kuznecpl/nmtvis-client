Introduction

We will quickly introduce NMTVis and its main functions and components before you can start. NMTVis is an interactive machine-translation system for helping translators to translate large documents. The system takes a German document and translates it into English, sentence by sentence. Users can view the machine translations, and interactively correct them if necessary. The system improves its translations using these corrections.

Note

You can leave this tab open while you use the system and use it as a reference to read the relevant parts when needed.

Main Functions

These are the main functionalities supported by NMTVis:

  • Translate large, domain-specific documents
  • Find incorrect translations through metrics and filtering
  • Interactively correct translations
  • Adapt the system and improve translations using users' corrections

NMTVis is a visual analytics system. Multiple views and visualizations are used to support the user in their task.

Overview

Metrics View

What you need to know:

  • The Metrics View is used to filter and select sentences from the document based on metrics such as sentence length.
  • Each metric is represented by an axis and each sentence is a line that intersects each axis at the value of that sentence for that metric.
  • Example: Long sentences will intersect the Sentence Length metric at the top of the axis.
  • This is also called a parallel coordinates plot.
  • Sentences can be selected by hovering over a line or in the list below, and a highlighted sentence will appear as an orange line.
  • Blue lines represent currently visible sentences, while gray lines represent sentences that are currently filtered out.
  • Brushing (clicking and draggin up/down) an axis with the mouse creates a filter for the respective metric. Remove the filter by clicking anywhere else on the axis.
  • This filter can be dragged up and down, and multiple filters can be set.
  • Sentences can be sorted based on metrics, by clicking on the sort icon below each axis.
  • You can drag and drop axes to switch the order of metrics.

These are the metrics used in the Metrics View:

  • Confidence: The confidence of the system regarding a translation. High values suggest a good translation, low values suggest errors.
  • Sentence Length: The number of words in the source sentence. This is useful as long sentences may contain errors.
  • Keyphrases: The number of keyphrases, i.e. domain-specific words, in a sentence. This metric is important to ensure such words are translated correctly.
  • Document Order:The index of a sentence in the document, e.g. the first sentence has index 0.
  • Coverage Penalty:This metric indicates if words from the source sentence were not translated. A high penalty may indicate translation errors.

Keyphrase View

What you need to know:

  • The Keyphrase View shows a list of keyphrases, which are domain-specific words, sorted based on their frequency.
  • The number next to each keyphrase indicates the number of current sentences that the keyphrase appears in.
  • Keyphrases can be activated by clicking on the checkbox next to them.
  • By activating a keyphrase, only sentences containing the keyphrase are displayed below.
  • All active keyphrases are highlighted in the source sentence on the left, as well as their translation on the right.
  • If a keyphrase was mistranslated by the system, this should be visible by looking at highlighted words.
  • New keyphrases can be added by entering them in the text field below the list of keyphrases.

Beam Search View

What you need to know:

  • The Beam Search View shows you all translations generated by the system as a tree.
  • The number of translations shown can be changed at the top.
  • Each circle represents a word, and links between words connect the words of a translation.
  • The color of a circle represents the probability of system assigned to the related word.
  • Red circles signify likely words, while light circles signify unlikely words.
  • An orange ring around a circle highlights the currently selected word.
  • Following the tree from the SOS circle to an EOS circle, a complete translation is created.
  • Orange links represent the current best translation, which you can change by clicking on any other EOS circle.

Navigation and Interaction

  • By moving from the SOS to a EOS circle, you complete a translation.
  • Hover over a node to select a (partial) translation.
  • You can move through the translations using ← → ↑ ↓ keys as seen in the figure.
  • Press ENTER or click to show alternative words to continue a translation.
  • Type any key to create a custom correction, if you do not like any alternatives shown.
  • Press BACKSPACE to remove the last character from your custom correction or move backwards.
  • You can also use TAB to move forwards through a translation.

Attention View

What you need to know:

  • The Attention View visualizes how words in the source sentence relate to words in the current translation.
  • Lines with varying thickness connect source words with translation words.
  • The thicker the line, the more important the attached source word was for translating the word in the translation.
  • On the other hand, if there is no line from a source word to any word in the translation, this might suggest that the word was not properly translated.
  • During interaction with the Beam Search View, the Attention View highlights the relevant source words to aid in correcting a translation.
  • You can also hover over words in the source sentence and translation, which highlights all related words in the respective other language.

Your Task

Once you click on the Start Now button below, a new tab to the main view of NMTVis will open. A document is already uploaded, so you can start right away. The document is a German Wikipedia article about Autonomous Driving. There is no time limit, and you are free to interact with the system in any way you like. There are 3 tasks you should perform using the system, in order to answer the questionnaire afterwards.

Task 1: Select Sentences

Try to browse through the list of sentences, and find incorrect machine translations. You can flag incorrect translations by clicking on the flag icon on the right. You can use the Metrics View to select or filter based on attributes such as length or confidence. Also, use the Keyphrase View to filter sentences that contain certain words that might be difficult to translate. When you are ready to correct a sentence, simply click on it in the list, you will be linked to a detailed view.

Task 2: Explore Translations

Try to explore translations made by the system using the Beam Search View. You can use your mouse to zoom and pan around the tree of translations. You can increase the number of translations using the size input element. Also, hover over the Attention View that will show you which German words the system focused on for translating an English word.

Task 3: Correct Translations

We now enter the expert mode: correcting a translation. In the Beam Search View try to navigate using your arrow keys. You can go left, right, up and down to move along through a translation. If you think a word is incorrect, press Enter to see alternatives or simply type the correct word that should follow and then navigate to that node. You can also zoom and pan using the mouse, and click or hover nodes.

Once your are finished using the system, please click Finish Study in the top right corner, where you will be asked a few questions regarding the system.
